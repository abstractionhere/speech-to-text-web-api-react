# Speech to text app using Browser API

## Download and install

#### Download
```shell
git clone https://github.com/abstractionhere/speech-to-text-web-api-react
cd speech-to-text-web-api-react
```

#### Installation
```shell
npm install
```

## Running the app

```shell
npm start
```

Open ```http://localhost:3000``` in your browser
